Flex布局——属性

十一月 30, 2016

简介

现在网页布局比较流行的是div布局，但是深有体会的是例如垂直居中就不是很容易实现，虽然网上给了好多方案，但都稍麻烦；相对来说，Flex布局就可以用简简单单几个属性就能完成各种各样的布局

有人可能会担心浏览器的支持
浏览器 支持的版本
IE 10+
FireFox 22+
Chrome 21+
Opera 12.1+
Safari 6.1+

属性

要使用Flex布局首先要建立一个Flex容器，即在任何一个元素上添加

<div class='container'<!-容器--
    <div class='item'</div<!-项目--
    <div class='item'</div
    <div class='item'</div
</div

.container{
    display:flex;
    display:-webkit-flex;//webkit内核浏览器需要添加的
}

整个容器有两个轴，主轴（默认水平，相当于x轴）和交叉轴（默认垂直，相当于y轴）

其他在容器上使用的属性

  1. flex-direction

    规定其子元素即项目的排列方式，规定了主轴的方向，因为后面的属性都以主轴为标准，所以会影响到后面的属性

    属性值

    • row: 主轴水平，从左到右
    • row-reverse: 主轴水平，从右到左
    • column: 主轴垂直，从上到下
    • column-reverse: 主轴垂直，从下到上

  2. flex-wrap

    flex-direction虽然规定了排列方式，但是如果元素超过了一行怎么办，so flex-wrap属性规定了项目如果超过一行，要如何换行

    属性值

    • nowrap: 强制排列在一行，宽度等分，这时width规定的宽度会失效
    • wrap: 自动换行，即正常的换行，第一行显示不完，自动换到第二行从左到右显示
    • wrap-reverse： 换行，但是这种是按行倒着排列，即假设有两行，则第二行显示在前面，第一行显示在第二行下面

  3. flex-flow

    这个属性就是把flex-direction和flex-flow写在一起，例如 flex-flow: row wrap-reverse

  4. justify-content

    规定了项目整体在主轴上的排列方式，即一行一行地

    属性值（假设在主轴水平从左到右的情况下）

    • flex-start: 每一行项目靠左
    • flex-end: 每一行项目靠右
    • center: 每一行项目居住靠拢
    • space-between: 每两个项目中间间距相同，左右两端项目紧挨容器边界
    • space-around: 例如某一行3个项目，则每一行的剩余宽度等分成6份，每个项目两侧两份，且不共用

  5. align-content

    规定了项目整体在交叉轴上的排列方式

    属性值（假设在主轴水平从左到右的情况下）

    • flex-start: 整体靠上
    • flex-end: 整体靠下
    • center: 垂直居中
    • space-between: 第一行靠顶端，最后一行靠底端，中间每两行间距相同（即 将剩下的高度等分）
    • space-around: 假如有3行，则将除了每一行的高度剩余的高度等分成6份，每一行上下两份且不共用

  6. align-items

    规定了每个项目在交叉轴上的位置

    属性值

    • flex-start: 靠上
    • flex-end: 靠下
    • center: 垂直居中
    • stretch: 项目高度拉伸占满整个父元素的高度
    • baseline: 项目的第一行文字的基线对齐。

在项目上使用的属性

  1. order

    规定了元素的顺序（order规定的顺序值可以打乱html中的结构中的顺序）

    属性值： 就是一个整数值，越小越靠前

  2. flex-grow

    规定主轴上一行的元素的放大比例，即对剩余空间的分配，如果一行地元素都没有设置宽度，那么就是将这一行的宽度按照flex-grow的值进行比例分配

    属性值：同样也是数值

  3. flex-shrink

    与flex-grow对应，规定了元素的缩小比例，默认值为1，即父元素的宽度减小的时候，这些元素的宽度等比例缩小

    属性值：数值（大于等于0，决定缩小比例，一个项目该属性值为0的时候，不进行缩小）

  4. flex-basis

    规定了一个项目的宽度，和width效果基本相同

  5. flex

    这是上面三个属性flex-grow、flex-shrink、flex-basis的简写

  6. align-self 规定了该项目在交叉轴上的位置，可以覆盖align-items

    属性值

    • flex-start: 靠上
    • flex-end: 靠下
    • center: 垂直居中
    • stretch: 项目高度拉伸占满整个父元素的高度
    • baseline: 项目的第一行文字的基线对齐。
    • auto: 这个是指该项目的交叉轴上的位置由父元素的align-items决定
